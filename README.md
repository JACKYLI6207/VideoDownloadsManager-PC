# Video Downloads Manager PC

Windows 桌面版影片下載管理工具。內建瀏覽器擴充用於媒體嗅探，本機下載引擎負責 HLS / MP4 下載與合併。

## 功能概要

| 模組 | 說明 |
|------|------|
| 瀏覽器 | 透過 Google Chrome 與內建擴充嗅探可下載的媒體資源 |
| 進行中 | HLS 雙進度條（下載 / 合併）、批量操作、任務匯入匯出 |
| 已完成 | 下載歷史與開啟檔案 |
| 日誌 | 執行紀錄 |
| 設定 | 並行數、路徑、選用第三方擴充 |

### 下載引擎

- HLS 多執行緒片段下載，邊下邊合併
- 內建 FFmpeg，輸出標準 MP4
- 大檔 HTTP Range 並行下載
- 403 / 401 / 429 自動暫停，可手動繼續
- 任務持久化，重開程式可還原

## 系統需求

- Windows 10 / 11（64 位元）
- Google Chrome（未安裝時，首次啟動瀏覽器功能會提示自動安裝）
- 首次使用瀏覽器 / 合併功能可能需要網路（Chrome 安裝、Chromedriver、下載來源）

## 下載執行檔

請至本儲存庫 **Releases** 頁面下載 `VideoDownloadsManagerPC.exe`（單一檔案，約 78 MB）。

使用方式：下載後直接執行，無需安裝 Python。

## 從原始碼建置

```powershell
python -m venv .venv
.\.venv\Scripts\pip install -r requirements.txt
powershell -ExecutionPolicy Bypass -File build_exe.ps1
```

產出：專案根目錄 `VideoDownloadsManagerPC.exe`

## 專案結構

```
├── extension-src/     # PC 版瀏覽器擴充原始碼
├── vdm_pc/            # Python 主程式（UI、下載引擎、橋接服務）
├── scripts/           # 建置與輔助腳本
├── main.py            # 程式入口
├── vdm-pc.spec        # PyInstaller 設定
├── requirements.txt
└── logo.ico
```
